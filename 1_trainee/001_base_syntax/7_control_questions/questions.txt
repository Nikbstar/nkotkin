    2. К какому типа языка программирование относиться Java?

    3. Из каких компонентов состоит Java (JDK, JRE, JVM)?
        - виртуальная машина +
        - клиентская чась
        - либо часть для разработки

    4. Для чего используется JDK?
        для разработки и запуска программного обеспечения на java.

    5. Для чего используется JRE?
        для запуска программного обеспечения, разработанного на JDK.

    6. Для чего используется VM?
        Чтобы программы java работали на любой операционной системе, на которой можно запустить jvm.

    7. Расскажите про примитивные типы.
        Примитивный тип - это тип данных, предоставляемый языком программирования как базовая встроенная единица языка.
        В Java таких типов 8:
        boolean byte short int long char float double
        эти типы данных хранят в себе значение.

   8. Что такое ссылочные типы?
        ссылочные типы данных хранят в себе только ссылки на объекты.

   9. Что такое классы обертки?
        это примитивные типы данных, представленные как ссылочные.

    10. Что такое автобоксинг и анбоксинг?
        автоматическое свертывание и разветрывание примитивных типов данных в ссылочные.

    11. Как в Java передаются параметры в методы?
        аргументы передаются по значению. (объекты представляют ссылки, поэтому если изменить объект в методе, то изменится и основной объект.

    12. Почему нельзя изменить ссылку на объект в методе?
        потому что в метод передается не ссылка а объект, который содержит копию ссылки.

    14. Что такое оператор условия?
        это оператор, при котором, в зависимости от условия выполняется то или иное действие.

    15. Какие типы оператор условия существуют?

        switch

    16. Почему нельзя сравнивать ссылочные типы по оператору ==?
        потому что при этом будут сравниваться не объекты, а ссылки на эти объекты.

    17. Почему примитивные типы можно сравнивать через оператор ==?
        потому что в примитивных типах хранятся значения.

    18. Какие объекты могут быть использованы в операторе switch?

        стринги
        енумы

    19. Расскажите про булевы операции || && ? Что такое short-circuit?
        операции И и ИЛИ необходимы чтобы задавать сложные условия для условных операторов и циклов.
        short-circuit - идет проверка первого условия, и если оно false, то нет смысла проверять второе условие.

    20. Что такое тернарное условие?
    max = (a > b) ? a : b;
    условный оператор if в одной строчке.

    21. Что такое циклы и для чего они используются?
        циклы используются для многократного повторения определенного участка кода.

    22. Для чего используется цикл for?
        в нем задается начальнео значение, конечное значение и шаг.
        цикл используется для того чтобы повторить какой то код определенное количество раз.

    23. Для чего используется цикл foreach?
        с помощью итератора перебирает элеметны массивов и листов.

    24. Для чего используется цикл while?
        в этом цикле задается только условие выхода из него.
        используется чтобы повторить код неопределенное количество раз (пока не выполнится условие).

    25. Для чего используется цикл do while?
        в этом цикле сначала идет выполнение тела цикла, а затем проверка условия.
